About The Project

A task manager API that can be used as a boilerplate for similarly structured APIs. The features that the API provides are the following:

  • User authentication.
  • Email notifications for opening and closing an account.
  • User and task route authorization.
  • User and tasks CRUD operations.
  • Tasks pagination and sorting.

Built With

  • NodeJS
  • ExpressJS
  • MongoDB
